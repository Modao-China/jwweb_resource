# jwweb_resource
青果教务系统虚拟机系统分享~ (已破解安装)

已经制作成vbox虚拟机磁盘镜像, 仅供学习\
链接: https://pan.baidu.com/s/1dGBUy9F 密码: myu9

大二下学期期末的时候折腾的, 现在才发, 纯粹就是提醒一下自己曾经还是有点geek的~

包含一个web系统安装时候注册机(思路: 直接工.net反编汇工具找解密代码, 提取那个函数出来, 就可以了)

在安装服务端数据库, 应用服务器等需要另外一个密匙, 通过Ollydbg下断点, 然后就是漫长的按下一步, 等那个密匙字符串显示出来, 看了52破解一个入门教程就开干了, 找到密匙的时候心疼学校花了100+w(听说)买这个教务系统

用老的软件来入门这方面还是挺不错的
